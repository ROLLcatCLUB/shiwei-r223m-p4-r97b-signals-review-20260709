# R223M-P4 与 P3 对照

| 项目 | P3 | P4 |
| --- | --- | --- |
| 教师稿形态 | 连续稿 + 师维控制点 | 连续稿 + 师维控制点 + 小教判断短摘要 + 派生物影响摘要 |
| R97B 信号 | 未系统吸收 | 抽取正文第一、影响摘要、确认门、低权重小教、页边注规则 |
| 组件呈现 | 后端 JSON 中完整记录 | 主稿只写影响摘要，完整触发仍在标准文档 |
| 派生物 | 大屏、学习单、证据触发沉到结构 | 每环节以 1 行摘要说明会影响什么 |
| 确认门 | 边界文字存在 | 明确定义 preview_only=true / teacher_confirmed=false / formal_apply_allowed=false |
| UI 边界 | formal_ui=blocked | 继续不改 R97B、不做 UI、不接 runtime |

P4 不是功能扩展，也不是页面实现。它是把 R97B 静态页已经证明有效的产品秩序并入课堂事件展开标准。
